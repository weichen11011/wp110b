# 專題報告

## 動機
在一次觀看影片時，推薦欄跳出了用react的寫法來撰寫俄羅斯方塊，好奇心的驅使下便點了進去，然後也開始跟上影片的步驟
自己來嘗試做俄羅斯方塊。

## 過程
使用react的途中，一大堆沒看過的新東西讓我有點頭暈，不過還是可以理解影片的步驟裡的邏輯想法。但第一次嘗試做出來的框架，格子數明顯與俄羅斯方塊中的12X20不同，
也花了很多時間來處理，由於問題一直無法解決所以最後決定重新使用三個檔案.css,.html,.js的撰寫方式。
